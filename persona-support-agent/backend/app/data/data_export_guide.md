# Data Export Guide

## Requesting an Export
1. Go to Account Settings > Data Export.
2. Choose format: CSV or JSON.
3. Select date range and data types (users, transactions, logs).
4. Click "Request Export". You will receive a download link via email within 24 hours.

## Export Limits
- Free/Pro plans: 1 export request per 7 days.
- Enterprise: unlimited export requests.

## Large Exports
Exports exceeding 5GB are split into multiple files and delivered as a zipped archive link valid for 7 days.
